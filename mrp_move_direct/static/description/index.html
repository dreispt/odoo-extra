mrp_move direct Module

Add or Cancel Raw Material in Manufacturing Orders

By Elico Corp

This module allows for modification to be done directly to the consumed item in a manufacturing order. Users will be able to completely scrapped an item or add a new item that will be used in the manufacturing process of the finish product without having to change the bill of materials. 

Features:
- Add new raw material item to be consumed in the MO
- Cancel/remove raw material item to be consumed in the MO 

This module also allows for users to completely scrap an item in Stock Moves.
